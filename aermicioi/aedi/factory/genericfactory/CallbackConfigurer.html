    Class CallbackConfigurer

    Encapsulates aggregate configuration logic using a delegate.

    Encapsulates aggregate configuration logic using a delegate. The algorithm calls delegate, with a locator a set of Args, and configured aggregate, in order to modify the aggregate somehow.

    Template arguments

    class CallbackConfigurer(T, Dg, Args...)
    if (is(T == class) && (is(Dg == void delegate(Locator!(), T, Args)) || is(Dg == void function(Locator!(), T, Args))));
